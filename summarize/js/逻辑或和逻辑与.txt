---------------------------------------
&&:逻辑与

true = true && true;
flase 一假即假;


var result = a && b;
如果a为真，则返回b;
如果a为假，则返回a;

------------------------------------

||:逻辑或

false = false || false;
true 一真即真;

var result = a || b;
如果a为真，则返回a;
若果a为假，则返回b;

-------------------------------------
&&喜欢假的，||喜欢真的。

javascript中 0、”“、null、false、undefined、NaN为假。
